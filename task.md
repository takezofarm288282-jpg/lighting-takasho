# 商品詳細モーダル実装

## やること
1. LIXIL各商品ページから複数画像URLをスクレイピング
2. TAKASHO各商品ページから複数画像URLをスクレイピング
3. schema.ts に `images` カラム追加（JSON配列）
4. seed.ts に各商品の複数画像を追加
5. DB push & seed実行
6. selector.tsx に詳細モーダルUI実装（画像スライダー+説明）
7. GitHub push

## 詳細モーダル仕様
- 商品画像クリックで開く
- 画像スライダー（複数枚）
- 商品名・型番・価格・スペック
- 照射シーン・取付写真・角度図など
